# All India Velama Association (AIVA) - North America Student & Career Portal

A modern, simple members-only community portal where AIVA students and working professionals can share journey blogs, post open roles, and add contact details (email only) so others can reach out for opportunities or help.

## Features

- Journey blogs: Publish personal career/learning stories with contact email
- Roles board: Post internship, full-time, part-time, or contract roles
- Reach-out directory: Add support area with email-only contact
- Smart filtering: Search blogs, roles, and helper contacts instantly
- Modern UI: Responsive glassmorphism-style cards and clean layout

## Project Structure

- `index.html` - Portal layout and sections
- `styles.css` - Responsive modern styling
- `script.js` - Client-side data rendering, posting, and search
- `package.json` - Minimal run script for local server
